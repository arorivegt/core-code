Create a webpage that prints the numbers 1 - 10, each number being a different color. (No JS/TS)

## Solution
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>WEEK-8 - Tuesday - CoreCode Bootcamp</title>
  </head>
  <body>
    <ul style="background-color: rgba(85, 80, 80, 0.794);">
      <li style="color: red;">1</li>
      <li style="color: blue">2</li>
      <li style="color: yellow;">3</li>
      <li style="color: yellowgreen;">4</li>
      <li style="color: brown;">5</li>
      <li style="color: aqua;">6</li>
      <li style="color: blueviolet;">7</li>
      <li style="color: rgb(176, 197, 192);">8</li>
      <li style="color: white">9</li>
      <li style="color: blueviolet;">10</li>
    </ul>
  </body>
</html>
```
